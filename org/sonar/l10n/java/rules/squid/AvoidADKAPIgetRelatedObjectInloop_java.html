PLM rule to avoid ADK method getRelatedObject in Loop

Noncompliant Code for getRelatedObject Example


for (int j = 0; j < theSims.size(); j++) {
	
	Map theNextMap = (Map)theSims.get(j);
	getRelatedObject();
}
