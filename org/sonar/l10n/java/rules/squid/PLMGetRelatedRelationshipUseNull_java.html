@PLM Rule : Use Null instead of new stringList as argument in getRelated API.

Noncompliant Code for getRelated Example


Pass Null value as argument instead of new StringList() for getRelatedObjects/getRelatedObject/expandSelect method.


DomainObject dom = new DomainObject();
dom.getRelatedObjects(context, String, String, new StringList(), new StringList(), boolean, boolean, Level, String, String, 0); 
dom.getRelatedObject(context, new StringList(), boolean, String, new StringList());
dom.expandSelect(context, String, new StringList(), new StringList(), new StringList(), boolean, boolean, Level, String, String, 0);

Compliant Solution for get info

Sample code snippet would look like this.
Applicable to getRelatedObject API.

dom.getRelatedObjects(context, String, String, null, null, boolean, boolean, Level, String, String, 0);
dom.getRelatedObject(context, null, boolean, String, null);
dom.expandSelect(context, String, null, null, new StringList(), boolean, boolean, Level, String, String, 0);
