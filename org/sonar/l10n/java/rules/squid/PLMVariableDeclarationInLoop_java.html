PLM rule to avoid Variable declaration in Loop

Noncompliant Code for Variable Declaration in Loop Example


for (int i =0;i<10;i++)
{
	int a = 0;
	String b = "";
}

above Variable declarations cane be done outside of a loop as shown below.

Compliant Solution for Variable Declaration in Loop

Sample code snippet would look like this.
int a;
String b;
for (int i =0;i<10;i++)
{
	a = 0;
	b = "";
}
